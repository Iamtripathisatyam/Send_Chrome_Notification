We are going to see how to send a Chrome notification from your PC to a phone or several devices in this article. We may send messages, links, and other content. For this, we'll use Python's ***notify2*** module and its various capabilities, which will help us in delivering messages to many devices.

### Installation: 
```python
pip install notify2
```
